PLAKAR-TOKEN(1) - General Commands Manual

# NAME

**plakar-token** - Manage Plakar tokens

# SYNOPSIS

**plakar&nbsp;token**
\[**create**]

# DESCRIPTION

The
**plakar token**
command manages tokens used to authenticate to Plakar services.
Tokens are not currently usable and exist only for future features.

# SUBCOMMANDS

**create**

> Create a new token.
